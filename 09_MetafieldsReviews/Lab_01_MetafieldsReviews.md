# Lab - Metafields and Product Reviews

### Prerequisites:

* BigCommerce store (sandbox or live)
* Basic knowledge of APIs
* REST client (Postman)
* An existing Postman environment and collection as configured in previous labs

## Metafields Exercises

Try the following requests in the "Metafields" folder.

### Add Product Metafields

* **Enter** a valid product ID for the `:product_id` query param.
* **Observe** the metafield response data.

### Get Product Metafields

* **Enter** the same product ID for the `:product_id` query param.
* **Observe** the metafield response data.

## Reviews Exercises

Try the following requests in the "Product Reviews" folder.

### Create Product Review

* **Enter** a valid product ID for the `:product_id` query param.
* **Observe** the review response data.

### Get Product Reviews

* **Enter** the same product ID for the `:product_id` query param.
* **Observe** the review response data.
